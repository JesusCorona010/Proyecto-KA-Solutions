## 📌 About the Project

This is the **first version** of the official website for KA Solutions. It was designed and developed as a professional corporate site to showcase the company's services, team, and contact information.

The site features a modern dark theme, animated logo, and a clean responsive layout built entirely with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

> ⚠️ **Note:** This version has been taken offline while **v2.0** is in development.

## 📄 Pages

- **Home** — Hero section with animated logo and call to action
- **About Us** — Company background and mission
- **Services** — Digital solutions offered by KA Solutions
